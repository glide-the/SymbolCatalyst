## 二、数据流设计

### 主流程：角色选择 → 完整游戏循环

```
1. 玩家选择角色卡
   ↓
2. 【角色解析器】提取：性格向量、动机优先级、禁忌边界、规则偏置
   ↓
3. 【世界观适配器】匹配角色特质 → 动态生成世界观骨架
   ├─ 势力分配（基于角色动机冲突）
   ├─ 资源分布（基于角色优势/弱点）
   └─ 价值观冲突轴（基于角色禁忌）
   ↓
4. 【剧情生成器】构建3幕式叙事
   ├─ 第一幕：角色动机启动事件
   ├─ 第二幕：禁忌被测试的冲突升级
   └─ 第三幕：规则偏置触发的终局抉择
   ↓
5. 【抉择构建器】在关键节点生成选项
   ├─ 每个选项标注：短期收益、长期代价、隐藏变量变化
   └─ 基于角色性格过滤不合理选项
   ↓
6. 【现实事件库】随机或触发注入现实变量
   ↓
7. 【变量映射器】将现实事件映射为游戏数值
   ↓
8. 【动态改写器】基于新数值重写规则卡文本
   ↓
9. 玩家做出决策
   ↓
10.【风险计算器】计算失败概率 + 【代价结算器】执行后果
   ├─ 成功：推进剧情 + 解锁新规则卡
   ├─ 失败：扣除信誉/资源 + 可能触发「规则污染」（规则永久变难）
   └─ 特殊：触发「规则偏置」（角色专属规则改写）
   ↓
11.【后果追踪器】更新所有变量 + 记录选择分支
   ↓
12.【存档系统】保存状态 → 返回步骤4继续循环
```

### 辅助流程：规则动态改写

```
触发条件（任一）：
├─ A. 现实事件注入（如：「政府出台新法规」）
├─ B. 角色规则偏置触发（如：「背叛者角色遭遇信任危机」）
├─ C. 玩家主动宣言（如：「我公开挑战这个规则」）
└─ D. NPC行为干预（如：「权势NPC改写法律」）
   ↓
【动态改写器】执行：
├─ 1. 提取当前规则卡的「可改写字段」
├─ 2. 基于触发源计算改写方向（强化/削弱/反转/添加例外）
├─ 3. 生成新规则文本 + 保持原规则作为「历史版本」
├─ 4. 【平衡性校验】检测是否破坏游戏性
│   ├─ 如破坏：生成反制卡或代价修正
│   └─ 如合理：推送规则变更通知
└─ 5. 【规则污染度】+1（污染度越高，后续规则越不稳定）
   ↓
输出：新规则卡推送至玩家界面
```

## 三、模块技术规格

### 1. 角色解析器
- **输入**：角色卡JSON（性格/动机/禁忌/优势/弱点/规则偏置）
- **输出**：性格向量（8维坐标）、动机优先级队列、禁忌权重矩阵
- **算法**：基于特质心理学（Big Five + Dark Triad）的向量化

### 2. 剧情生成器
- **输入**：角色向量 + 世界观骨架 + 当前剧情状态
- **输出**：3幕结构 + 分支树 + 关键抉择点列表
- **模板库**：200+剧情原型（英雄之旅/悲剧/复仇/救赎等）

### 3. 规则引擎
- **卡牌数据结构**：
  ```json
  {
    "id": "rule_001",
    "name": "信誉契约",
    "type": "社会秩序",
    "trigger": "当做出承诺时",
    "effect": "获得资源×2",
    "cost": "信誉度-10，若违约则双倍扣除",
    "counter_card": "rule_045",
    "modifiable_fields": ["effect", "cost"],
    "pollution_level": 0
  }
  ```
- **改写算法**：基于模板填充 + 语义保持（确保规则逻辑自洽）

### 4. 现实秩序映射器
- **12维变量定义**：每个变量0-10取值 + 影响权重
- **映射函数库**：50+现实事件 → 变量变化规则
- **安全性约束**：所有映射经过伦理审查（不映射违法内容）

### 5. 记忆与存档
- **短期记忆**：当前会话的所有抉择 + 变量快照
- **长期记忆**：跨赛季的角色成长树 + 规则污染度累积
- **玩家画像**：基于历史行为的偏好模型（用于推荐新角色/世界观）

## 四、性能与扩展性

- **响应时间**：抉择生成 < 2秒，规则改写 < 1秒
- **并发支持**：1000+玩家同时在线
- **扩展接口**：
  - 新角色卡：JSON格式导入
  - 新世界观：填空模板 + 参数配置
  - 新规则卡：卡片编辑器（UGC支持）
  - 现实事件源：API接入新闻/社交媒体（需过滤）

## 五、商业化模块挂载点

```

```

---

**架构总结**：这是一个「叙事-规则-现实」三位一体的生成系统，核心创新在于**现实秩序映射层**，它让游戏不再是逃避现实的幻想，而是理解现实的实验室。
