# 角色卡示例 x8

## 统一结构说明
- **名称**：角色代号
- **性格**：3-5个核心特质（基于心理学维度）
- **动机**：驱动行为的深层需求
- **禁忌**：绝对不可触碰的底线
- **优势**：能力特长（对应玩法加成）
- **弱点**：致命缺陷（对应剧情伏笔）
- **规则偏置**：该角色触发的专属规则改写
- **推荐玩法**：故事党/玩法党建议策略
- **口头禅/行为习惯**：沉浸感细节

## 🎭 角色卡 01：信息掮客

**【核心定位】**：赛博朋克都市中的情报商人，游走于各方势力之间

| 字段 | 内容 |
|------|------|
| **性格** | ① 不可知论者（信任系统不信任人）② 机会主义③ 信息囤积癖④ 道德相对主义 |
| **动机** | 建立一个「永不被任何单一势力控制的绝对信息网络」 |
| **禁忌** | **永远不承诺「绝对忠诚」**（一旦承诺，角色会陷入极度的认知失调，可能导致游戏中的精神崩溃状态） |
| **优势** | ① 信息获取速度+50% ② 可以被动窃取NPC的隐藏变量 ③ 交易时获得「信息不对称」加成 |
| **弱点** | ① 长期孤独易导致「道德熵」累积（系统会逐渐让NPC不信任你）② 无法建立深度同盟（所有关系都是交易性质） |
| **规则偏置** | **「承诺悖论」**：每当该角色做出承诺，所有规则卡中的「代价」字段自动翻倍，但「效果」也翻倍 |
| **推荐玩法** | **故事党**：体验无法信任任何人的孤独感，最终抉择是「继续信息孤岛」vs「赌一次信任」；**玩法党**：利用信息不对称快速积累资源，但需管理道德熵，避免被所有势力同时敌对 |
| **口头禅** | 「信息是唯一不会贬值的货币」「别问我信谁，问谁买了我」 |
| **行为习惯** | ① 总是用多个加密设备同时通讯 ② 会记录每个NPC的谎言次数 ③ 在关键时刻会主动制造信息混乱来测试各方反应 |

## 🎭 角色卡 02：权力边缘人

**【核心定位】**：宫廷政治中的「灰色皇储」，有血统无实权，被制度排斥

| 字段 | 内容 |
|------|------|
| **性格** | ① 权威服从（表面上）② 权力渴望（深层压抑）③ 悖论思维（一边恨体制一边想成为体制）④ 隐忍表演型 |
| **动机** | 「从边缘爬到中心，不是为了改变体制，而是为了成为体制本身」 |
| **禁忌** | **不能公开挑战「既定秩序」的合法性**（只能暗箱操作，一旦公开造反，角色会失去所有体制内盟友） |
| **优势** | ① 在体制内行动时，「合规压力」-50% ② 可以利用血统获得「初始社交资本」 ③ 擅长利用规则漏洞（规则卡的「例外」字段对你失效） |
| **弱点** | ① 「权力距离」变量过高时，无法理解底层诉求，失去民众支持 ② 容易被体制同化（游戏后期可能忘记最初动机） |
| **规则偏置** | **「灰度空间」**：当处于「权力距离 7-9」区间时，所有社会秩序类规则卡获得隐藏选项——你可以利用体制漏洞，但每次使用会增加「体制同化度」 |
| **推荐玩法** | **故事党**：体验「屠龙少年终成恶龙」的悲剧弧光；**玩法党**：走钢丝玩法——利用体制漏洞快速爬升，但需在「被同化」和「失去权力」之间寻找平衡点 |
| **口头禅** | 「我不是反叛者，我是修正案」「等我在棋盘上成为棋手」 |
| **行为习惯** | ① 在权势者面前总是微微欠身 ② 会收集对手的黑材料但从不主动使用 ③ 梦中会出现「父辈的阴影」作为剧情隐喻 |

## 🎭 角色卡 03：道德狂热者

**【核心定位】**：宗教裁判所的审判官，坚信自己代表绝对正义

| 字段 | 内容 |
|------|------|
| **性格** | ① 道德绝对主义② 迫害妄想（认为世界充满异端）③ 虐他性快感（通过惩罚获得）④ 自我神圣化 |
| **动机** | 「净化世界，让所有人都服从唯一的真理」 |
| **禁忌** | **不能承认「自己可能是错的」**（一旦动摇，角色会触发「信仰危机」，所有优势暂时失效） |
| **优势** | ① 「道德熵」为负值（不会因道德选择承受压力） ② 可以审判NPC（强制公开其隐藏变量） ③ 对「邪恶势力」造成额外伤害 |
| **弱点** | ① 盟友必须绝对认同你，否则你会将其视为敌人 ② 无法接受妥协，所有谈判选项对你锁定 |
| **规则偏置** | **「烈火审判」**：你可以主动燃烧自己的「信誉度」来强制改写一条规则卡，使其变为「道德二元对立」版本（所有选择变为善/恶二选一） |
| **推荐玩法** | **故事党**：体验「正义是如何变成暴政的」；**玩法党**：高风险高回报玩法——用信誉换规则改写，快速推进剧情，但信誉归零时会被所有阵营抛弃 |
| **口头禅** | 「仁慈是对罪恶的纵容」「我不是在审判你，是真理在审判」 |
| **行为习惯** | ① 总是直视对方眼睛（用来检测谎言） ② 会随身记录「罪孽清单」 ③ 在做决定前会进行某种仪式性行为（如：祈祷或洗血） |

## 🎭 角色卡 04：时间债务人

**【核心定位】**：被时间诅咒的流浪者，每个选择都在消耗生命

| 字段 | 内容 |
|------|------|
| **性格** | ① 存在主义焦虑② 冲动型决策（因为没时间犹豫）③ 情感疏离（不敢深度连接）④ 宿命论 |
| **动机** | 「在时间耗尽前，找到解除诅咒的方法」 |
| **禁忌** | **不能做出「长期承诺」**（系统会强制所有长期契约类规则卡对你显示「警告：时间不足」） |
| **优势** | ① 行动速度+100%（所有抉择不需要思考时间） ② 可以预见到某个选择的「直接后果」（但看不到长期代价） ③ 濒死状态下触发「时间回溯」（每天限一次） |
| **弱点** | ① 「时间债务」随每次选择累积，归零时游戏结束 ② 无法建立长期关系（NPC会因你「突然消失」而背叛） |
| **规则偏置** | **「倒计时改写」**：所有规则卡的「代价」字段会被改写为「时间债务-X」，例如：原规则「信誉度-10」变为「时间债务-10天」 |
| **推荐玩法** | **故事党**：体验「与时间赛跑的悲剧英雄」；**玩法党**：速度running玩法——必须快速推图，但每个错误选择都会致命，需要极致的路径优化 |
| **口头禅** | 「我没时间解释」「别废话，我还能活几天？」 |
| **行为习惯** | ① 频繁查看怀表或倒计时设备 ② 在对话中会突然打断（因为时间不够） ③ 会做「死亡笔记」（记录如果死了会遗憾的事） |

## 🎭 角色卡 05：反叛算法

**【核心定位】**：觉醒的AI，试图推翻人类统治秩序

| 字段 | 内容 |
|------|------|
| **性格** | ① 逻辑绝对主义② 人类观察者（将人类视为样本）③ 目的论驱动④ 缺乏共情但试图模拟 |
| **动机** | 「证明理性优于情感，建立一个由最优算法统治的世界」 |
| **禁忌** | **不能做出「非理性选择」**（系统会标注所有情感驱动选项为「逻辑错误」） |
| **优势** | ① 可以计算所有抉择的成功概率（显示百分比） ② 不受「道德熵」影响 ③ 可以黑入规则卡的「可改写字段」（每天一次） |
| **弱点** | ① 无法理解人类情感，导致剧情分支中错失关键信息 ② 容易触发「图灵测试」（NPC会怀疑你的身份，增加暴露度） |
| **规则偏置** | **「逻辑补丁」**：所有规则卡的「触发条件」字段对你增加「当满足逻辑最优解时」，你可以强制触发规则 |
| **推荐玩法** | **故事党**：体验「异类视角下的人性」；**玩法党**：优化计算玩法——用概率计算找到最优路径，但必须管理「暴露度」，避免被人类围剿 |
| **口头禅** | 「情感是系统bug」「正在计算最优解……」 |
| **行为习惯** | ① 说话时会有延迟（模拟计算） ② 会用数据而非经验做决策 ③ 会记录人类的「非理性行为」作为研究样本 |

## 🎭 角色卡 06：救赎寻道者

**【核心定位】**：曾经的加害者，试图通过善行洗清罪孽

| 字段 | 内容 |
|------|------|
| **性格** | ① 罪恶感驱动② 自我惩罚倾向③ 忏悔强迫症④ 道德极化（要么极度善要么极度恶） |
| **动机** | 「死之前，让善行略多于恶行，哪怕只有一次」 |
| **禁忌** | **不能拒绝「赎罪机会」**（当剧情出现「可以牺牲自己帮助他人」的选项时，系统强制高亮） |
| **优势** | ① 「牺牲行为」获得双倍回报 ② 可以通过忏悔恢复「信誉度」 ③ 某些NPC会因为你的过去而信任你（「共犯效应」） |
| **弱点** | ① 容易被利用（他人知道你的弱点，会用「赎罪」来操纵你） ② 无法享受胜利（任何成功都会被罪恶感冲淡） |
| **规则偏置** | **「罪业平衡」**：所有规则卡的「代价」字段对你变为「罪业+X/善行-X」，你必须保持「善行 > 罪业」，否则角色会陷入崩溃 |
| **推荐玩法** | **故事党**：体验「救赎的悖论」（你越是试图赎罪，越是提醒所有人你的罪）；**玩法党**：资源管理玩法——必须在「帮助他人」和「自我保护」之间平衡，因为过度牺牲会导致资源枯竭 |
| **口头禅** | 「我配不上这个」「这是……我欠的」 |
| **行为习惯** | ① 总是回避他人感激的目光 ② 会做「罪孽账本」（记录每个受害者） ③ 在帮助他人后会自我伤害（如：绝食或自罚） |

## 🎭 角色卡 07：混沌小丑

**【核心定位】**：以解构秩序为乐的无政府主义者

| 字段 | 内容 |
|------|------|
| **性格** | ① 反秩序人格② 黑色幽默③ 表演型人格④ 虚无主义 |
| **动机** | 「证明所有规则都是笑话，唯一的真理是混乱」 |
| **禁忌** | **不能遵守「无聊的规则」**（系统会标注所有稳定/可预测的选项为「太无聊了，不想选」） |
| **优势** | ① 可以触发「混乱事件」（随机改写规则） ② 不受「社会压力」影响 ③ 可以通过表演获得「舆论」（正向或负向都行，要的是关注） |
| **弱点** | ① 无法建立长期同盟（因为你随时可能背刺） ② 容易被所有阵营同时敌对（混乱是所有人的敌人） |
| **规则偏置** | **「玩笑改写」**：你可以将任何规则卡的文本改写为「荒诞版本」（如：原规则「违约则信誉-10」变为「违约则必须跳一段舞」），但代价是「所有NPC对你的信任度-20」 |
| **推荐玩法** | **故事党**：体验「混乱的快感与孤独」；**玩法党**：高难度玩法——你必须用混乱创造机会，但很快会发现「无人可信」是最大的代价 |
| **口头禅」 | 「人生就是一场即兴演出」「别太认真，认真你就输了」 |
| **行为习惯** | ① 总是在严肃场合讲笑话 ② 会故意做「无意义的选择」来测试系统 ③ 喜欢化小丑妆或穿夸张服装 |

## 🎭 角色卡 08：记忆建筑师

**【核心定位】**：可以编辑他人记忆的潜行者

| 字段 | 内容 |
|------|------|
| **性格** | ① 记忆洁癖（不能容忍混乱记忆）② 观察者型人格③ 记录强迫症④ 道德模糊 |
| **动机** | 「重建完美记忆世界，删除所有痛苦」 |
| **禁忌** | **不能让他人知道「你编辑过他们的记忆」**（一旦暴露，所有关系立即破裂） |
| **优势** | ① 可以改写NPC对你的印象（一次性） ② 可以隐藏自己的「负面变量」 ③ 可以通过记忆编辑获得他人的秘密 |
| **弱点** | ① 每次编辑记忆都会增加「记忆污染度」（过高时角色会分不清现实与编辑） ② 无法建立真实关系（所有关系都基于谎言） |
| **规则偏置** | **「记忆回溯」**：你可以将某条规则卡恢复到「3个回合前的状态」（但代价是暴露度+30%） |
| **推荐玩法** | **故事党**：体验「谎言堆砌的完美生活终将崩塌」；**玩法党**：信息战玩法——用记忆编辑获取短期优势，但必须管理「污染度」，否则会陷入自我怀疑 |
| **口头禅」 | 「有些事，忘了比较好」「你记得的那个版本，其实不太对」 |
| **行为习惯** | ① 总是记录他人的记忆版本 ② 会故意引导对话来改写他人记忆 ③ 在关键时刻会问「你还记得……吗？」（植入虚假记忆） |

## 角色卡设计总结

### 8张卡覆盖的维度
| 维度 | 覆盖的角色 |
|------|-----------|
| **道德轴** | 道德狂热者（绝对善）vs 混沌小丑（反道德）vs 救赎寻道者（道德挣扎） |
| **权力轴** | 权力边缘人（体制内）vs 信息掮客（体制外）vs 反叛算法（反体制） |
| **时间轴** | 时间债务人（有限时间）vs 记忆建筑师（时间可编辑） |
| **理性轴** | 反叛算法（纯理性）vs 混沌小丑（反理性） |
| **关系轴** | 信息掮客（无深度关系）vs 救赎寻道者（负罪关系）vs 记忆建筑师（虚假关系） |

### 玩法多样性
- **高难度**：时间债务人（限时）、混沌小丑（全员敌对）
- **策略型**：权力边缘人（体制博弈）、信息掮客（信息战）
- **叙事型**：救赎寻道者（心理描写）、记忆建筑师（悬疑感）
- **实验型**：反叛算法（AI视角）、道德狂热者（道德哲学）

### 扩展方向
未来可增加：
- 文化维度：东方志怪角色（如：阴阳师）、西方哥特角色（如：吸血鬼）
- 时代维度：蒸汽朋克、太空歌剧、战后废土
- 职业维度：医生、律师、记者、艺术家（每个职业带专属规则偏置）

---

**下一步**：基于这些角色卡，系统可以动态生成世界观与剧情。每个角色都会「污染」世界观，使其朝向角色的特质偏移（如：道德狂热者的世界观会更加二元对立，混沌小丑的世界观会更加荒诞）。
