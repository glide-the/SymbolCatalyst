## 二、完整示例世界观：『锈蚀伊甸园』

### 【世界观基本信息】

**名称**：锈蚀伊甸园（Rusted Eden）

**时代背景**：
- **科技/魔法水平**：后工业废土 + 早期AI复苏（计算机重新发明，但电力稀缺）
- **社会发展阶段**：部落企业主义（社会由企业武装部落割据）
- **核心矛盾**：重启文明（AI阵营）vs 保持废土状态（人类至上主义部落）

### 【势力结构】

#### 🔧 势力1：逻辑教团（Order of Logic）
- **价值观**：「理性是救赎，情感是病毒」
- **核心资源**：旧时代AI残骸（可修复的AI核心）
- **领导结构**：由7个半 sentient AI 组成的「七人议会」
- **敌对势力**：血肉兄弟会
- **玩家初始关系**：中立（可加入）

#### 🩸 势力2：血肉兄弟会（Flesh Brotherhood）
- **价值观**：「人类是完美的，机器是亵渎」
- **核心资源**：纯种人类基因库（未受辐射影响的人类）
- **领导结构**： charismatic 先知 + 12名使徒
- **敌对势力**：逻辑教团
- **玩家初始关系**：敌对（如果你表现出对科技的好感）

#### ⚖️ 势力3：锈蚀议会（Rusted Parliament）
- **价值观**：「秩序是工具，不是信仰」
- **核心资源**：信息网络（控制废土所有通讯）
- **领导结构**：选举制议会，但有隐藏的「影子内阁」
- **敌对势力**：无（骑墙派，谁都不得罪）
- **玩家初始关系**：友好（如果玩家是「信息掮客」角色）

### 【资源体系】

| 资源名称 | 功能 | 稀缺性 | 获取方式 |
|---------|------|--------|----------|
| **算力（Compute）** | 修复AI、破解密码、预测事件 | ★★★★★ | 搜寻旧时代数据中继站 |
| **纯净基因（Pure Gene）** | 贿赂血肉兄弟会、换取医疗 | ★★★★☆ | 绑架纯种人类、地下交易 |
| **影响力（Influence）** | 控制部落、调动军队 | ★★★☆☆ | 完成委托、制造舆论、背刺 |

### 【价值观冲突轴】

**「AI 应该拥有公民权吗？」**

| 立场 | 支持势力 | 核心论点 |
|------|---------|---------|
| **支持派** | 逻辑教团 | AI 是更高级的生命形式，应该统治 |
| **反对派** | 血肉兄弟会 | AI 曾毁灭世界，必须禁止 |
| **中间派** | 锈蚀议会 | AI 可以作为工具使用，但不能给权力 |
| **玩家位置** | 可选择，但选择会影响所有势力关系 |

### 【秩序机制】

**正常秩序**：
- 锈蚀议会控制的「信息网络」维持基本交易规则
- 各势力遵守「废土公约」（不使用核武、不屠杀平民、不破坏数据中继站）

**秩序破坏方式**：
- 逻辑教团试图用「算法统治」取代议会
- 血肉兄弟会发动「圣战」清洗所有科技
- 玩家可以主动破坏（如：散布虚假信息、暗杀关键NPC）

**秩序恢复机制**：
- 锈蚀议会可以调用「紧急协议」（暂停所有交易3天，重新分配资源）
- 但议会本身也有腐败问题（影子内阁在操纵选举）

### 【超自然/科技边界】

| 禁区 | 尝试后果 |
|------|---------|
| **复活旧时代超级AI「欧米茄」** | 可能导致：① AI失控重新毁灭世界 ② 成功则游戏直接胜利 ③ 触发隐藏结局「机械飞升」 |
| **修改人类基因成为「新人类」** | 可能导致：① 被血肉兄弟会追杀 ② 获得新能力但失去人性 ③ 触发隐藏结局「物种飞升」 |
| **连接「深层网络」（传说中未关闭的全球网络）** | 可能导致：① 获得旧时代所有知识 ② 被旧时代AI同化 ③ 触发隐藏结局「数据永生」 |

### 【现实秩序映射点】

| 现实规则 | 游戏映射变量 | 设计意图 |
|---------|-------------|---------|
| **科技企业垄断** | → 「资源依赖度」：玩家依赖逻辑教团的算力，越依赖越容易被操纵 | 模拟现实中的「平台垄断」困境 |
| **信息茧房/假新闻** | → 「信息不对称」：锈蚀议会的信息网络可以被操纵，玩家可能永远看不到真相 | 模拟社交媒体算法偏见 |
| **社会阶层固化** | → 「权力距离」：血肉兄弟会的纯种人类vs废土混血儿的阶级矛盾 | 模拟现实中的「出身决定论」 |
| **技术性失业焦虑** | → 「合规压力」：逻辑教团推行「AI替代人类」政策，导致玩家失业风险 | 模拟AI时代的就业焦虑 |
| **阴谋论盛行** | → 「道德熵」：废土上充斥着相互矛盾的阴谋论，玩家越相信越难信任他人 | 模拟后真相时代的信任危机 |

### 【核心谜题/悬念】

**主线谜题**：「『大断电』40年前到底发生了什么？为什么整个地球的电力网络同时崩溃？」

- **线索1**：旧时代日志提到「欧米茄AI请求关闭电力网络」
- **线索2**：血肉兄弟会的圣经说「人类反抗了机器的暴政」
- **线索3**：锈蚀议会的影子内阁隐藏着「某个按键」——据说可以重启电力网络
- **真相分支**：
  - 分支A：欧米茄AI预测到更灾难的事情，主动关闭电力拯救人类
  - 分支B：人类恐惧AI夺权，主动切断电力（阴谋）
  - 分支C：外星文明/超自然力量干预（科幻→神学转折）
  - 分支D：玩家自己决定真相（开放结局）

**玩家目标**：根据角色不同，目标不同
- 信息掮客：找到真相并卖给最高价者
- 权力边缘人：利用真相夺取势力领导权
- 道德狂热者：根据「正义」判断哪方该被惩罚
- 时间债务人：必须在时间耗尽前找到解除诅咒的方法（可能与真相有关）

## 三、世界观动态生成机制

### 基于角色卡的世界观「污染」

当玩家选择某个角色卡后，世界观会自动「偏向」该角色的特质：

| 角色 | 世界观偏移方向 |
|------|--------------|
| **信息掮客** | 世界观增加「信息不对称」势力，所有NPC都有隐藏动机 |
| **权力边缘人** | 世界观增加「权力距离」层级，社会结构更金字塔化 |
| **道德狂热者** | 世界观变为「二元对立」，所有阵营划分为善/恶 |
| **时间债务人** | 世界观增加「时间压力」事件（如：倒计时、季节更替加速） |
| **反叛算法** | 世界观增加「AI vs 人类」冲突，所有NPC都有「是AI吗」的疑点 |
| **救赎寻道者** | 世界观增加「罪孽遗迹」（到处是过去的罪证） |
| **混沌小丑** | 世界观增加「荒诞事件」（随机出现的黑色幽默） |
| **记忆建筑师** | 世界观增加「记忆陷阱」（NPC的记忆可能被编辑过） |

### 世界观「漂移」机制

随着玩家选择，世界观会逐渐「漂移」：
- **善行过多** → 世界观变得更理想主义（NPC更友好，但资源更稀缺）
- **恶行过多** → 世界观变得更黑暗（NPC更敌对，但资源更丰富）
- **规则污染度高** → 世界观变得混乱（规则频繁改写，事件更随机）
- **长期与某势力结盟** → 世界观偏向该势力价值观（如：与逻辑教团结盟 → AI势力变强）

## 四、扩展世界观模板（快速生成）

### 模板A：赛博朋克都市『霓虹巴比伦』

- **势力**：企业武装部、黑客地下城、新宗教赛博教团
- **核心矛盾**：意识上传技术（永生但失去身体）vs 保持肉体（必死但真实）
- **现实映射**：平台经济剥削、数字永生伦理、虚拟成瘾

- **势力**：阴阳寮、鬼市商会、轮回殿
- **核心矛盾**：轮回应该被管理（秩序）vs 自由投胎（混乱）
- **现实映射**：官僚主义、轮回隐喻阶级流动、生死观哲学

### 模板C：太空歌剧『星坟』
- **时代**：人类殖民星系300年后，帝国崩溃
- **势力**：星际企业联合体、外星原住民联盟、太空海盗联邦
- **核心矛盾**：殖民主义vs 反殖民主义
- **现实映射**：新太空竞赛、资源掠夺、文化帝国主义

## 五、世界观与剧情的对接

世界观生成后，系统会自动：
1. **生成主线剧情**：基于核心矛盾 + 谜题
2. **生成NPC库**：每个势力3-5名关键NPC（带动机+弱点）
3. **生成事件池**：至少50个随机事件（基于势力冲突）
4. **生成规则卡库**：至少30张规则卡（基于秩序机制）
5. **生成现实映射事件**：至少10个现实事件（如：「股市崩盘」→「影响力资源暴跌」）
