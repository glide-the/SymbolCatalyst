# 规则卡示例 x12

## 统一结构说明

```
┌────────────────────────────────────────────────────────────┐
│  【规则卡名称】                                              │
│  类型：______  触发：______  效果：______                    │
│  代价：______  反制：______  可改写字段：______              │
│  污染等级：__/10（越高越不稳定）                             │
└────────────────────────────────────────────────────────────┘
```

**字段解释**：
- **类型**：社会秩序 / 风险管理 / 剧情推进 / 反制机制 / 现实映射
- **触发**：什么条件下激活这张卡
- **效果**：激活后发生什么
- **代价**：使用这张卡的资源/数值消耗
- **反制**：哪张卡可以克制这张卡
- **可改写字段**：哪些字段可以被动态改写（如：效果、代价、触发条件）
- **污染等级**：0-10，越高表示这条规则越不稳定（容易被改写或产生意外后果）

## 📜 规则卡 01：信誉契约（社会秩序类）

| 字段 | 内容 |
|------|------|
| **名称** | 信誉契约 |
| **类型** | 社会秩序 |
| **触发** | 当你向NPC做出「正式承诺」时 |
| **效果** | 立即获得资源 ×2（基于目标NPC的信任度） |
| **代价** | 承诺期间「信誉度」锁定，若违约则扣除信誉 ×3 + 所有NPC对你的信任度-20% |
| **反制** | 「信息掮客」角色可以用「谎言检测器」看到你的违约概率 |
| **可改写字段** | 效果倍数、违约代价、资源类型 |
| **污染等级** | 2/10（稳定，但会被某些角色改写） |
| **卡牌文本** | *"承诺是一种债务。签字吧，或者滚出去。" —— 锈蚀议会的契约模板* |

**改写示例**：
- 被「权力边缘人」改写：效果变为 ×3，但违约代价变为「被体制永久排斥」
- 被「混沌小丑」改写：代价变为「违约时必须跳一段舞」（荒诞版）

## 📜 规则卡 02：信息不对称（现实映射类）

| 字段 | 内容 |
|------|------|
| **名称** | 信息不对称 |
| **类型** | 现实映射 |
| **触发** | 当你与NPC交易时，且你的「信息获取度」> NPC |

| **可改写字段** | 优化幅度、依赖度增长率、信息茧房阈值 |
| **污染等级** | 4/10（中风险，长期使用有害） |
| **卡牌文本** | *"我知道你不知道的。这就是利润。" —— 信息掮客的笔记本* |

**现实映射意图**：模拟现实中的「内幕交易」「信息茧房」「算法偏见」

## 规则卡设计总结

### 12张卡覆盖的功能维度

| 类型 | 卡牌 | 功能 |
|------|------|------|
| **社会秩序类** | 信誉契约、权力距离 | 模拟现实契约与阶层 |
| **现实映射类** | 信息不对称、合规压力、权力距离 | 映射信息差、监管风险、阶层固化 |
| **风险管理类** | 时间债务、道德熵 | 模拟时间压力与道德沦丧 |
| **剧情推进类** | 承诺悖论、烈火审判、灰度空间 | 角色专属机制 |
| **反制机制类** | 逻辑补丁、记忆回溯、玩笑改写 | 平衡过强规则 |

### 规则卡的「反脆弱」设计

每张规则卡都遵循「高风险高回报，失败有代价」：
- **信誉契约**：高回报（×2资源）但违约代价巨大（×3信誉）
- **信息不对称**：短期优势但长期茧房化
- **玩笑改写**：极强改写能力但全员敌对

### 规则卡的「动态改写」可能性

所有规则卡都有「可改写字段」，改写来源：
1. **角色偏置触发**（如：信息掮客触发「承诺悖论」）
2. **现实事件注入**（如：「新法规出台」→ 合规压力规则改写）
3. **玩家主动宣言**（如：「我挑战这个规则」→ 进入决斗模式）
4. **NPC干预**（如：权势NPC改写法律）
5. **规则污染度累积**（污染度 > 5 则规则开始随机变异）

## 下一步：规则与剧情的融合

这些规则卡会在剧情的关键抉择点出现：
- **抉择前**：系统显示当前适用的规则卡
- **抉择时**：玩家可以选择「使用规则卡」或「违反规则」（违反有风险但可能触发隐藏分支）
- **抉择后**：系统计算代价 + 规则可能被改写

示例：
> 你面对血肉兄弟会的先知，他要求你交出AI核心。
>
> **当前适用规则**：
> - [信誉契约]：你可以承诺明天交出（效果：获得时间×2 | 代价：若违约则信誉-30）
> - [合规压力]：你可以拒绝并开战（效果：保留AI核心 | 代价：通缉度+50）
> - [权力距离]：你无法直接对话，必须通过中间人（效果：对话选项-50% | 代价：社交资本-70）
>
> **你的选择**：______
