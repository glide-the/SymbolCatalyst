## 🎯 核心设计亮点

### 1. 三位一体架构
```
叙事引擎 ───── 规则引擎 ───── 现实秩序映射层
   ↓              ↓              ↓
故事党获得    玩法党获得    「临游戏」核心
情感张力      可计算博弈    创新体验
```

### 2. 角色设计的内在矛盾
每张角色卡都有「动机 vs 禁忌」的矛盾：
- **信息掮客**：想要信任 vs 不能承诺
- **权力边缘人**：想改变体制 vs 不能公开挑战
- **道德狂热者**：想拯救世界 vs 不能承认自己可能错
- **时间债务人**：想建立关系 vs 没有时间
- **救赎寻道者**：想赎罪 vs 越赎罪越被提醒罪孽

### 3. 规则卡的反脆弱设计
所有规则卡遵循：**高风险高回报，失败有代价**
- 信誉契约：×2资源，但违约×3代价
- 信息不对称：短期优势，长期茧房化
- 玩笑改写：最强改写，但全员敌对

### 4. 现实秩序映射的12维变量
| 现实规则 | 游戏映射变量 | 示例影响 |
|---------|-------------|---------|
| 科技企业垄断 | → 资源依赖度 | 越依赖越容易被操纵 |
| 信息茧房 | → 信息不对称 | 长期使用看不到真相 |
| 阶层固化 | → 权力距离 | 向上流动需要更高代价 |
| 监管套利 | → 合规压力 | 违规有短期优势但长期风险 |

### 5. 三种商业模式互补
```
UGC市场（快速验证）
      ↓
赛季订阅（提升ARPU）
      ↓
B2B培训（稳定现金流）
```
